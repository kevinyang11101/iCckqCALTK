# 大数据应届生求职系统 python1021

## 项目概述

大数据应届生求职系统是一个基于Python语言的Web应用，集成了Django框架和Vue.js技术。系统致力于为应届生与招聘公司提供一个高效、便捷的求职与招聘平台。

## 技术栈

- **后端：** Python 3.7，Django框架
- **前端：** Vue.js
- **数据库：** MySQL
- **开发工具：** PyCharm

## 功能模块

- **首页：** 展示系统概览及重要通知
- **个人中心：** 用户资料管理与个性化设置
  - 求职者管理：管理个人求职信息
  - 面试邀请管理：查看与处理面试邀请
  - 面试结果管理：查看面试结果
- **公司管理：** 管理公司基本信息
  - 公司信息管理：维护公司资料
  - 招聘信息管理：发布与管理招聘信息
  - 投递简历管理：接收与筛选求职简历
  - 面试邀请管理：发送面试邀请
  - 面试结果管理：反馈面试结果
- **通知公告管理：** 发布与浏览系统通知
- **系统管理：** 系统设置与权限控制

## 系统角色

- **求职者：** 提交简历，管理求职进度
- **招聘公司：** 发布职位，管理招聘流程
- **系统管理员：** 维护系统运行，管理用户与权限

## 运行环境

- Python 3.7+
- Django 2.x
- Vue.js 2.x
- MySQL 5.x

## 部署步骤

1. 安装Python 3.7及其依赖
2. 安装MySQL数据库，配置数据库连接
3. 克隆项目代码到本地
4. 安装Django框架和Vue.js
5. 运行前端构建命令
6. 运行后端迁移命令
7. 启动开发服务器
